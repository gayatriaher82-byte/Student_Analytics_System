# Student_Analytics_System

Basic student record management system developed using Python and NumPy.

**By Gayatri Bajirao Aher**

## Project: Student Analytics System

**Objective:** Build a menu-driven application to manage student records and analyze student marks using Core Python and NumPy.

### Modules:
- Add new student records.
- View all student records.
- Update student information.
- Delete student records.
- Analyze marks using NumPy (sum, mean, median, standard deviation, minimum, maximum).
- Generate random student datasets.
- Filter pass/fail and topper students.
- Apply grace marks using NumPy.
- Perform matrix operations module (under development).

### Learning Outcomes:
This project demonstrates the practical application of Python fundamentals, lists, dictionaries, functions, CRUD operations, and NumPy for statistical analysis in a menu-driven application.
